# Página web de aniversario - 9 años ❤️

## Cómo usar en Visual Studio Code

1. Abre la carpeta `web_aniversario_9_anos` en Visual Studio Code.
2. Instala la extensión **Live Server**.
3. Clic derecho sobre `index.html`.
4. Selecciona **Open with Live Server**.

## Cómo subirlo a GitHub Pages

1. Crea un repositorio en GitHub.
2. Sube todos los archivos de esta carpeta.
3. Ve a **Settings > Pages**.
4. En **Branch**, selecciona `main` y carpeta `/root`.
5. Guarda.
6. GitHub te dará un link. Ese link lo conviertes en QR.

## Música

La página ya tiene botón de música.
Para que funcione:

1. Crea esta carpeta si no existe: `assets/music`
2. Coloca tu canción en formato MP3.
3. Nómbrala exactamente: `nuestra-cancion.mp3`

Ruta final:
`assets/music/nuestra-cancion.mp3`

Importante: GitHub Pages puede reproducir música solo después de que la persona toque el botón, por seguridad del navegador.


## Si no suena la música

Revisa estos puntos:

1. El archivo debe estar en:
   assets/music/nuestra-cancion.mp3

2. El nombre debe ser exactamente:
   nuestra-cancion.mp3

3. En Windows puede pasar que el archivo quede como:
   nuestra-cancion.mp3.mp3

   Para revisar eso:
   - Abre el Explorador de archivos
   - Ve a Vista
   - Activa "Extensiones de nombre de archivo"
   - Verifica el nombre real

4. Abre la página con Live Server.
   No la abras solo con doble clic.

5. La música no puede sonar automáticamente.
   La persona debe tocar el botón "Play música 🎵".
